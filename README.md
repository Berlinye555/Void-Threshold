# 虚空 · 门槛 — Void Threshold

> 一个暗黑科幻互动叙事项目。17 篇交织的故事碎片，来自 2098 年的未来。

---

## 这是什么

**虚空·门槛** 是一个科幻设定宇宙。它不是一个线性故事，而是一组互相关联的叙事碎片——人物日志、通信记录、数字展品、互动选择——散落在 2072 年到 2098 年之间。

你可以随意点开一篇，像在废墟里捡到一张泛黄的纸片，读完了再找下一张。慢慢拼出这个世界的全貌。
